## Create MASTER token (HOT exercise #1)

For this first exercise, we will be creating a MASTER token via Dynatrace UI that will be used for token management.

1. In Dynatrace tenant, go to Settings >> Access Tokens and "Generate new Token"

    ![settingsView](../../assets/images/generateToken.png)

2. Click on *Generate token*

3. Enter: 
    * Token name: MASTER
    * Search for TOKEN MANAGEMENT scope
    * Check TOKEN MANAGEMENT checkbox
    * Click on GENERATE TOKEN


    ![settingsView](../../assets/images/generationToken.png)

4. Copy the new API token generated and store it somewhere (ie. notepad)

    ![token](../../assets/images/token.png)
